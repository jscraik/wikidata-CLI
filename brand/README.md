# brAInwav Brand Assets

This folder contains approved documentation mark assets for use in repository READMEs.

## Usage
- Use the mark only in root-level README footer signatures.
- Keep the mark small (24-32px height) and left-aligned.
- Do not use as a watermark in technical docs.

## Files
- `brand-mark.webp`
- `brand-mark@2x.webp`
- `brand-mark.png` (optional fallback)
- `brand-mark@2x.png` (optional fallback)
