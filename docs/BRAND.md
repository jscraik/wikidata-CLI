## Quickstart
### 1) Use the brand palette
Primary colors:
- Dark: `#141413`
- Light: `#faf9f5`
- Mid Gray: `#b0aea5`
- Light Gray: `#e8e6dc`

Accent colors:
- Orange: `#d97757`
- Blue: `#6a9bcc`
- Green: `#788c5d`

### 2) Use the brand typography
- Headings: Poppins (fallback Arial).
- Body: Lora (fallback Georgia).

## Common tasks
### Add branded callouts in Markdown
- What you get: subtle brand styling without HTML.
- Steps:
  1. Use headings with clear, short titles.
  2. Use bold text for key actions only.
  3. Keep code blocks minimal and consistent.

## Troubleshooting
### Symptom: fonts do not render as expected
Cause: Poppins and Lora are not installed.
Fix: install fonts locally or rely on fallbacks.
